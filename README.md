# Back Bay Digital – Frontend Recreation

This project is a responsive frontend recreation of the Back Bay Digital website built using pure HTML, CSS, and JavaScript.

⚠️ Disclaimer:  
This is an independent educational and portfolio project. It is not affiliated with or endorsed by Back Bay Digital. All branding and content belong to their respective owners.

---

## Project Overview

The goal of this project was to:

- Practice advanced layout structuring
- Improve responsive design skills
- Recreate modern agency-style UI
- Strengthen HTML/CSS architecture
- Implement clean, scalable frontend code

---

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)

---

## Features

- Fully responsive layout
- Smooth scrolling
- Clean navigation structure
- Optimized section spacing
- Modern agency-style UI design
